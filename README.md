## 📖 Project Overview
This healthcare analytics project focuses on monitoring and optimizing pharmacy operations. By analyzing sales data across zones, products, and sales representatives, the dashboard provides actionable insights into regional performance and revenue drivers.

## 🎯 Strategic Questions Answered
- **Zone Efficiency:** Which geographical areas (e.g., Abbasia, Ain Shams) lead in sales volume?
- **Personnel Performance:** How do individual Area Managers and Sales Reps compare against performance benchmarks?
- **Product Portfolio:** Which medications (like Glucovance) are the primary revenue contributors?
- **Pricing Trends:** How do price fluctuations correlate with sales amount over time?

## 🛠️ Technical Implementation
- **Data Source:** Comprehensive pharmacy records including distributor details, pricing, and regional metadata.
- **Data Engineering:** Used **Power Query** for data cleaning, currency normalization, and merging lookup tables.
- **UI/UX Design:** Designed a clean, grey-themed interface with intuitive slicers for seamless exploration.

## 💡 Key Insights
- **Regional Dominance:** Identified specific zones that contribute over 40% of the total revenue.
- **Sales Rep Analysis:** Ranked representatives by efficiency, revealing high-performing individuals for talent recognition.
- **Product Demand:** Spotted seasonal trends in medication sales, aiding in inventory planning.
